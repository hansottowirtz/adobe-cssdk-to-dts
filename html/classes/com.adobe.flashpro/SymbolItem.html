public function exportToPNGSequence(outputURI:String, startFramNum:int, endFrameNum:int, jsMatrix:mathObject):void

Export an symbol item (movie clip, graphic symbol or button) in Symbol Library to a series of PNG files

Parameters

outputURI:String — (String) The URI to export the PNG Sequence files to. This URI must reference a local file. Example: file:///c|/tests/mytest.png (with written permission)
 
startFramNum:int — (int) start frame number which will be exported. The default number is 1
 
endFrameNum:int — (int) end frame number which will be exported. The default number is 99999
 
jsMatrix:mathObject — (mathObject) User can add additional matrix to generate PNG sequence
