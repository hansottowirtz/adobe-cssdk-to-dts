elementsproperty 
elements:Array  [read-only]

An array of element objects. The order of elements is basically the order in of how they are stored in the FLA file. Note: Flash treats ungroup shapes as 1 element object. For example: if a frame contains 3 raw shapes, then elements.length in that frame returns 1. Group the shapes to individual group to work around this issue.
