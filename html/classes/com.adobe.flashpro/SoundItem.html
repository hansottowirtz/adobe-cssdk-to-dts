exportToFile()method
public function exportToFile(fileURI:String):Boolean

Exports SoundItem in Library to sound file specified in the fileURI parameter. The file extension in the URI determines the file format of the output file (either MP3 or WAV). When exporting sound items, you should check if the "soundItem.originalCompressionType" property is equal to"RAW." If this check is false, you can only export the file as MP3. (Optionally, you can try exporting as a WAV file, and if the function returns false, then try to export to MP3.)

Parameters

fileURI:String — (String) File URI for the exported sound file.

Returns
Boolean
