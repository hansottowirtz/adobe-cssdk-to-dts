linkageBaseClassproperty 
linkageBaseClass:String

Base class for AS3 library symbols, appears in the linkage dialog in the ui (and other dialogs that include the linkage dialog controls, like the symbol properties dialog). If the base class is the default for the symbol type (e.g. "flash.display.MovieClip" for movie clips, "flash.display.SimpleButton" for Buttons) then this will be the empty string "", and setting it to the empty string is the way to make it the default base class. When setting this, none of the checks performed by the linkage dialog are done (checks to make sure that the base class can be found in the FLA's classpath, checks to make sure class is auto-generated and not defined in the FLA's classpath, ensures AS3 is chosed in publish properties and symbol is exported).
