Classpublic class Path
InheritancePath com.adobe.csawlib.CSHostObject

A instance of a path object can be returned from the flash object. path = fl.drawingLayer.newPath(); The path object is used to define a sequence of straight line and/or curved line segments. Typically used only when creating extensible tools.

makeShape()method 
public function makeShape(bSupressFill:Boolean, bSupressStroke:Boolean, recalcFill:Boolean, recalcStroke:Boolean, forceFillForCompoundPath:Boolean, useEvenOddFillMode:Boolean, jsFrame:Object):void

Creates a shape on the Stage using the current stroke and fill settings. The path is cleared after creating the shape. makeShape has 2 optional Boolean arguments to supress the fill and or stroke of the resulting shape object. If the parameters are omitted, or set to false, the current values for fill and stroke are used. Typically used only when creating extensible tools.

Parameters

bSupressFill:Boolean — (Boolean) if true the shape is not filled.
 
bSupressStroke:Boolean — (Boolean) if true no stroke is applied to the shape.
 
recalcFill:Boolean — (Boolean)
