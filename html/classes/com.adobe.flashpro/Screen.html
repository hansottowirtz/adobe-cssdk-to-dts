Packagecom.adobe.flashpro
Classpublic class Screen
InheritanceScreen com.adobe.csawlib.CSHostObject

The Screen object represents a single screen in a slide or form document. The Screen object contains properties related to the slide or form. The array of all the Screen objects in the document can be accessed from screenOutline.screens.

forceSimple:Boolean

Enable/disable the children of the object to be accessible. This is equivalent to the inverse logic of "Make Child Objects Accessible" setting in the Accessibility Panel. forceSimple == true --> Make Child Object Accessible is unchecked. forceSimple == false --> Make Child Object Accessible is checked.
