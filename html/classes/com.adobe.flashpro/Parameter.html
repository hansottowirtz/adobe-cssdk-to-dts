Method Detail
insertItem()method
public function insertItem(index:int, name:String, value:String, type:String):void

If the Parameter is of type List, Object or Array, then the value property is an array. To insert a value to the array, you must use the insertItem method.

Parameters

index:int — (int) The index of where this item will be inserted into the List, Object or Array. If index is 0, it will go at the beginning of the list and the other items will be shifted higher. If index is > the list size, then the new item will be at the end of the array.
 
name:String — (String) The name of the newly inserted item. It is a required argument, but only used if the Parameter is of type Object.
 
value:String — (String) the value of the new item inserted.
 
type:String — (String) The type of item to insert.

public function removeItem(index:int):void

To remove an element of the List, Object or Array type of Screen or Component Parameter, use the removeItem method.

Parameters

index:int — (int) The index of the item to remove from the Screen or Component Property.
