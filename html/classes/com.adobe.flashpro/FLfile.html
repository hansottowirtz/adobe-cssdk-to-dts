setAttributes()method 
public function setAttributes(fileURL:String, strAttrs:String):Boolean

This function sets the system-level attributes of a particular file. returns true if the operation succeeded, false otherwise.

Parameters

fileURL:String — (String) The fileURL argument, which is expressed as a file:// URL, identifies the file for which you are setting the attributes.
 
strAttrs:String — (String) The strAttrs argument specifies the system-level attributes for the file that is identified by the fileURL argument. The following table describes valid attribute values and their meaning: R Read only (on Mac, read only is "locked", not the permission) W Writable (overrides R ) H Hidden (Windows only) V Visible (overrides H, Windows only) Acceptable values for the strAttrs string are R , W , H , V , RH , RV , WH , or WV . You should not use R and W together because they are mutually exclusive. If you combine them, R becomes meaningless, and the file is set as writable ( W ). You should not use H and V together because they are also mutually exclusive. If you combine them, H becomes meaningless, and the file is set as visible ( V ). If you specify H or V without specifying an R or W read/write attribute, the existing read/write attribute for the file is not changed. Likewise, if you specify R or W , without specifying an H or V visibility attribute, the existing visibility attribute for the file is not changed.

Returns
Boolean
