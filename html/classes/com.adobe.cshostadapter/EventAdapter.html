Classpublic class EventAdapter
InheritanceEventAdapter flash.events.EventDispatcher
Subclasses AIEventAdapter, PSEventAdapter

This class provides the base implementation for Creative Suite application-specific event adapter libraries.

This class should not be instantiated directly to communicate with a host application. Instead, use the application-specific subclasses of this class. For example, to register listeners for Photoshop events, use the PSEventAdapter class.

This class extends flash.events.EventDispatcher. For more information on its APIs, see the ASDoc for EventDispatcher.

pingPlugIn()method 
public function pingPlugIn(responseHandler:Function, timeout:Number = 5000):void

Sends a 'ping' message to the host adapter plug-in and dispatches the response to the specified handler.

A PingEvent is dispatched to responseHandler either when a response is received from the plug-in or after the specified timeout period.

This method provides a means of checking whether the required host adapter plug-in has been loaded by the host application, and therefore whether this library can be used to register event listeners.

Parameters

responseHandler:Function — A listener function to which the resulting PingEvent will be dispatched
 
timeout:Number (default = 5000) — The number of milliseconds to wait for a response from the host adapter plug-in before it is considered not to be loaded
