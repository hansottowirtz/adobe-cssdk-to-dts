Classpublic class AIEventAdapter
InheritanceAIEventAdapter EventAdapter flash.events.EventDispatcher

This class provides APIs for registering and unregistering event listeners for Illustrator events. It requires the AIHostAdapter Illustrator plugin to be installed in the host application in order for its method calls to have any effect.

Use the addEventListener and removeEventListener methods to add and remove event listeners for Illustrator events. The set of event types which may be listened for are defined as constants in the AIEvent class. See the documentation for each constant for information on when events of that type are dispatched.

For example, to listen for the opening a document in Illustrator, use:

AIEventAdapter.getInstance().addEventListener(AIEvent.DOCUMENT_OPENED, listener);

where listener is a method with the following signature:

function listener(event:AIEvent)

When a document is opened in Illustrator, an AIEvent instance with type AIEvent.DOCUMENT_OPENED will be dispatched.

Note that this library dispatches events asynchronously; Illustrator will continue to execute immediately after an event has been dispatched. This means, for example, that a listener for the AIEvent.DOCUMENT_ABOUT_TO_CLOSE event is not guaranteed to be executed before the document has been closed, despite the event's name. Therefore listening for such events using this library should be avoided.
