Classpublic class Layer
InheritanceLayer PhotoshopHostObject CSHostObject flash.utils.Proxy
Subclasses ArtLayer, LayerSet

A layer object. Layers may contain nested layers, which are called sublayers in the user interface. The layer object contains all of the page items in the specific layer as elements. Your script can access page items as elements of either the layer object or the document object.
