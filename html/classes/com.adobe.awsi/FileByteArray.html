Packagecom.adobe.awsi
Classpublic class FileByteArray

This class wraps a ByteArray and associated Filename for uploading to Adobe Web Services. The chunk size will be used to determine how many bytes will be uploaded each time this is used. The offset will set the position of the supplied Byte Array. If none is provided, it defaults to the start.
