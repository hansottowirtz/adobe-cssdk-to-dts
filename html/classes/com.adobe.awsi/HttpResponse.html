Packagecom.adobe.awsi
Classpublic class HttpResponse

This is the base class for dealing with AWSI responses. It parses the xml payload to give access to the following information:

  • root - provides access to the root of the response DOM (composed of ResponseNode's)
  • attributes - convenient access to the attributes on the root results element
  • results - provides access to the root's direct child nodes.
  • Attributes of the nodes can be further queried.
