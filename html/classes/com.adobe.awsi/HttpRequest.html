Classpublic class HttpRequest
InheritanceHttpRequest flash.events.EventDispatcher

A class for making HTTP requests to a specified target. It allows various request attributes to be specified:

  • HTTP method to use (e.g. GET/POST)
  • HTTP request headers and parameters
  • End point and path to target
  • HTTP library implementation to perform the request

The class abstracts over the process common to many Adobe Web Services of interpolating parameters into request end points and paths.

Parameters not interpolated into the end point or path are sent as HTTP parameters with the request.

HttpRequest()constructor
public function HttpRequest(method:String, endPoint:String, path:String)

Constructs an HTTP request object which will be performed using the specified HTTP method, targeting the specified end point server and path. Together, the end point and path should represent a complete target for the HTTP request being constructed:

  • The end point represents the URL of the server to which the request should be made. It may include the port number to use. For example, "http://www.myserver.com:80"
  • The path specifies the location on the end point to which the request should be made. Typically it begins with a forward-slash ('/'). For example, "/myservice"

Parameters
method:String — the HTTPMethodType to use (e.g. GET)
 
endPoint:String — the end point to target
 
path:String — the path on the specified end point

interpolate()method 
protected function interpolate(interpolatePath:String):String

Returns a new path with parameters that have been indicated with {param} being replaced by their value. Matching parameters are removed from the parameter map. A special case currently exists for the {.format} term. It will be mapped to the _format parameter. We may wish to generalize this so that any . prefixed placholder gets substituted with its _ prefixed equivalent parameter. === Example var params = {version: "v1.0", _format: "xml", other: "value"}; var path = "/{version}/albums{.format}"; interpolate(path, params) will return "/v1.0/albums.xml" Params will then be {other: "value"}

setOption()method 
public function setOption(key:String, value:Object):void

Sets additional information to be used by the request. For example, when using the FlexHttpClientLib additional option keys include:

  • unique_id - Attaches this to all events dispatched in the HttpLibrary and is returned to the onComplete function when the request is complete.
  • timeout - Sets the timeout limit for the request. The default is 90000 milliseconds
  • debug - If set to true debug output will be passed to the AwsiTracer class.
